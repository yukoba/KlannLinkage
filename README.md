# Klannのリンク機構
[Klann's Linkage Plans - DIY Walkers](https://www.diywalkers.com/klanns-linkage-plans.html) および [Klann linkage - Wikipedia](https://en.wikipedia.org/wiki/Klann_linkage) を元に [Linkage Mechanism Designer and Simulator](https://blog.rectorsquid.com/linkage-mechanism-designer-and-simulator/) で作成しました。

動画：[Klannリンク機構 - YouTube](https://www.youtube.com/watch?v=l21C2VVgt4A)
[![Klannリンク機構 - YouTube](http://img.youtube.com/vi/l21C2VVgt4A/maxresdefault.jpg)](https://www.youtube.com/watch?v=l21C2VVgt4A)

### レゴ化
それを [BrickLink Studio](https://www.bricklink.com/v3/studio/main.page) でレゴ化し KlannLinkage.io ファイルをレンダリングしました。
実物だと簡単に組み上げられるのですが BrickLink Studio では真ん中の緑のビームがちゃんと接するようにアプリ上で組み上げる方法が不明でした。

![KlaanLinkage.png](KlaanLinkage.png)

### MuJoCo化
[MuJoCo](https://mujoco.org/) に移植したものを [yukoba/KlannLinkageMuJoCo - GitHub](https://github.com/yukoba/KlannLinkageMuJoCo) に置きました。
